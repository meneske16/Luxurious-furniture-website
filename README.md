# Luxurious Furniture (Website)


**Live demo:** https://luxurious-furniture.netlify.app/


**Short description**
A responsive static website showcasing Luxurious Furniture's product collections with WhatsApp-based "View Details" inquiry buttons.


## Contents
- Homepage with Signature Collection
- Separate pages for Living, Dining, Bed, Bath, Outdoor, Decor, Bespoke creations
- Responsive layout with mobile-friendly hamburger menu
- WhatsApp inquiry buttons (linking to wa.me)


## Tech stack
- HTML5
- CSS3 (responsive design)
- Optional: Vanilla JavaScript for mobile menu and interactions
- Hosted on Netlify (static site)
